# Doubles Rotation Solver MVP

Static browser-only prototype for generating doubles schedules with fewer repeated partners and opponents.

## Run

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 4173
```

## Scope

- Paste player names.
- Choose rounds, courts, and optimization priority.
- Generate doubles games.
- Track repeated partners and opponents.
- Copy schedule, download CSV, or print.

## Not Included

- Accounts.
- Server storage.
- Online league management.
- Payments.
- Player profiles.
- Official tournament rule handling.

## Deployment

This is a static site. It can be deployed on a free static host subdomain for validation.
